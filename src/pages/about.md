# About

Bourdain Index is an open-source index and reading guide to Anthony Bourdain’s life and work.

It aims to answer what Bourdain wrote, filmed, said, visited, and appeared on; when it happened; where it can be read, watched, or heard legally; and how it connects to the rest of his work.

## What this is not

This project is not a content mirror, streaming site, transcript archive, or replacement for libraries, publishers, broadcasters, and other rights holders.

## Copyright and source policy

Bourdain Index does not host copyrighted books, full articles, full episode videos, or unauthorized transcripts.

It links to official sources, public archives, libraries, and source metadata.

Short excerpts may be included only when necessary for identification, commentary, or research context.

If a source is paywalled, the project should link to the canonical source and mark it as paywalled.

## Contributing

Contributions should favor source-backed metadata, original summaries, and stable links. Prefer adding a `needs-review` or `partial` entry over pretending uncertain metadata is confirmed.
