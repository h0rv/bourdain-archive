# Bourdain Index

## Run

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Data

One YAML file per entry under `src/content/`.

```yaml
id: kitchen-confidential
title: Kitchen Confidential
type: book
date: "2000"
date_precision: year
summary: Original short summary.
image_url: https://covers.openlibrary.org/b/isbn/9780060899226-M.jpg
tags: []
people: []
places: []
sources: []
related: []
status: needs-review
availability:
  official_url:
  archive_url:
  library_url:
  audio_url:
  video_url:
  transcript_url:
  streaming_url:
  purchase_url:
```

No book text. No article mirrors. No episode uploads. No transcripts.
