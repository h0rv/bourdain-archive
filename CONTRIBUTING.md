# Contributing

## Entries

- One YAML file per entry under `src/content/`.
- Lowercase hyphenated IDs.
- Original summaries only.
- No full text, mirrors, uploads, or transcripts.
- Use `needs-review` when unsure.

## Status

- `confirmed`
- `needs-review`
- `missing-source`
- `dead-link`
- `partial`

## Local

```sh
npm install
npm run dev
npm run build
```
